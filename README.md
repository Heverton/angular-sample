# Angular-Sample

Passo a passo para os estudos de Angula

## Step 1. Set up the Development Environment
npm install -g @angular/cli

## Step 2. Create a new project
ng new my-app

## Step 3: Serve the application
cd my-app
ng serve --open

## Criando componentes:
ng g c meu-segundo

## Criando modulos:
ng g m cursus
